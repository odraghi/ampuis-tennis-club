## Install Dev env
```
npm install -g grunt-cli bower yo generator-karma generator-angular
```

## Make a new route
```
yo angular:route myroute
```

## Build & development

Run `grunt --force` for building and `grunt serve` for preview.

## Testing

Running `grunt test` will run the unit tests with karma.
